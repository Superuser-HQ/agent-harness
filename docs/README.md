# Docs

## Structure

| Folder       | Purpose                          |
|--------------|----------------------------------|
| `prd/`       | Product Requirements Documents   |
| `adr/`       | Architecture Decision Records    |
| `research/`  | Research notes and findings      |

## Naming Convention

All documents follow the format: `[yyyy-mm-dd]-title.md`

Example: `prd/2026-02-16-agent-harness.md`
